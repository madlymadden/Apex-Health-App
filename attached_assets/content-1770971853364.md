#### The Equinox brand is bold, provocative and unapologetic. This distinctive tension is crafted through exceptional art direction and harnessed by Equinox Sans, a custom font created exclusively for Equinox.

#### The Problem

Equinox.com was initially designed using two fonts, Titling Gothic Compressed and Graphik, predating the introduction of Equinox Sans. A harmonious balance was struck between the bold display and sans-serif fonts. To maintain a consistent brand across all platforms, Equinox Sans would need to replace those fonts.

#### The Solve

Establish a harmonious balance by strategically employing Equinox Sans' weight, size, and placement. Systematically integrated within the existing web modules to serve as a foundational element of the visual language. In addition, the design team would require an updated design system with optimized components using the new font.

![](https://images.squarespace-cdn.com/content/v1/650a32ba207efd35b87c4854/179a773b-ac00-4628-8038-6b0cf2278383/eqxsans-collage-dt-final.jpg)

### Design System

A complete library swap was carried out after strategically replacing fonts and optimizing components using the latest features in Figma. The design team utilizes the system to efficiently scale existing designs and create new web experiences for both desktop and mobile devices.
